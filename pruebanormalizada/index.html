A

"Ä" o "ä" (Ltspkr.png Pronunciación), es un carácter que representa tanto una letra de varios alfabetos, como una letra A con umlaut o diéresis.

Esta página es sobre la letra latina be. Para la letra del alfabeto griego representada con un glifo similar, ver Β. Para la letra del alfabeto cirílico, ver В. Para otros usos de este término, véanse Be y B (desambiguación).
